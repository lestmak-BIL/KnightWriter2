## Description

v1.0 was shipped after a **critical-path smoke test** only. This issue tracks running the **full manual testing checklist** (Phase 4 from Testing-Plan.md) before or as part of v1.0.1.

**Scope:** Complete the checklist in `docs/MANUAL-TESTING-CHECKLIST.md` (sections 4.1–4.11): Core functionality, Source editor, Visual editor, Bidirectional sync, Format toolbar, Scroll sync, Keyboard shortcuts, Edge cases, **Accessibility**, **Cross-browser** (Chrome, Firefox, Safari, Edge), **Performance**.

**Out of scope for v1.0:** Accessibility audit, full cross-browser verification, and performance benchmarks were not run for v1.0. This issue closes that gap.

**Checklist:** See [docs/MANUAL-TESTING-CHECKLIST.md](../MANUAL-TESTING-CHECKLIST.md) in the repo. Copy its content into this issue (or link it) and check off each item as completed.
